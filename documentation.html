Advanced 404 to 301 Redirection Plugin Documentation

The Advanced 404 to 301 Redirection plugin for WordPress redirects 404 errors to specified URLs using 301 redirects and logs the redirections. This documentation provides information on installation, settings, log viewing, and frequently asked questions.

Installation

  1. Upload the advanced-404-to-301 folder to the /wp-content/plugins/ directory.
  2. Activate the plugin through the 'Plugins' menu in WordPress.
  3. Navigate to '404 to 301' in the admin menu to configure the redirection settings.

Settings

  1. Go to '404 to 301' in the WordPress admin menu.
  2. Enter the desired Redirect URL for 404 Errors.
  3. Click 'Save Changes' to save the settings.

Log

  1. Go to '404 to 301' in the WordPress admin menu and select 'View Log.'
  2. The log displays a history of 404 redirections.
  3. Use the 'Clear Log' button to clear the log.

FAQs

How do I set up a redirection URL?

Navigate to '404 to 301' in the admin menu, enter the desired Redirect URL, and save the changes.

How can I view the redirection log?

Go to '404 to 301' in the admin menu, select 'View Log,' and the log history will be displayed.

Can I clear the log?

Yes, you can clear the log by clicking the 'Clear Log' button in the 'View Log' section.
